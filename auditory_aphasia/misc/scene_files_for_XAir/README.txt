scene files for Behringer X-AIR audio device

- 6D
for auditory aphasia setup

- 6d_bass_boost
Same as 6D except equalizer setting for channel 7 and 8 (Channels for Headphone). Lower frequency range is boosted.

- Monitor
You can monitor all channel inputs from channel 7 and 8.

- naming_test
For naming test

- original_state_of_dcc
DCC default setting. You need to load this before you leave the room.
